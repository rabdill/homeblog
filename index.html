Projects

Hubbub DC

A database of DC happy hour specials searchable by date/time, Metro stop, and price of drinks. Written as an exercise in learning PHP and MySQL. HubbubDC.com.

TreasureMeasure

A personal budgeting application for people who hate personal budgeting. Intricately descriptive without being too complicated—it simulates your checking account months (or years) into the future, to help you better plan practical financial goals. Still in alpha testing.

AdventureNet

A tool for authoring hypertext-based interactive fiction. Written in PHP and MySQL, with a responsive design using customized Bootstrap stylesheets. Provides a GUI that allows the user to write a Choose Your Own Adventure-style computer game in a browser without any programming. Still in preliminary testing.

HomeBlog CMS

A simplified blogging system that integrates publishing into existing websites rather than generating new ones, as is the common practice of systems like Wordpress and Drupal. In progress.

Bishop's Map

A basic, text-based gaming engine written in Python that allows amateur programmers to prototype Zork-style interactive fiction games without having to first learn about Python objects and classes. Will also function as a basic Python tutorial. Still in progress.
